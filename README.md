# 🫧 Bubbles Game

A fun and interactive bubble-popping game built with **JavaScript** and **HTML**.

## 🎮 How to Play
* **Pop bubbles**: Click or tap on falling bubbles to destroy them.
* **Create bubbles**: Click on the background to spawn new bubbles manually.
* **Track stats**: Monitor the total number of shapes and their area in real-time.

## ✨ Features
* **Responsive**: Scales perfectly on both mobile and desktop.
* **Neon Design**: Vibrant colors with glow effects.
* **Fluid Animation**: Smooth gravity-based movement.

## 🛠️ Tech Stack
* **HTML**
* **CSS3**
* **JavaScript**
